2. Get to know the FHIR API structure

FHIR APIs are created from a standard framework of components called resources that are designed to be easy to implement. Resources are the building blocks of every request.

Read our guide to working with FHIR API resources

3. Learn about operations

Terminology FHIR APIs have a simple set of interactions like READ and UPDATE that follow a RESTful structure. This is extended by operations, which you use to actively formulate inputs and outputs.

Read our guide to making requests with FHIR API operations
