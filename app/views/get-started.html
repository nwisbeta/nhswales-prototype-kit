Get started

1. Registering interest in access

You can make a request to access the terminology service API if:

  • you work for an NHS Wales health board
  • you are a 3rd party organisation sponsored by an NHS Wales health board

Email terminology.service@wales.nhs.uk to register interest. In the email you should tell us the reason for your request, for example that you'd like to use the API to retrieve content from the Welsh Data Dictionary. This will help us to understand the best way to support you.

It may take up to 5 months to complete the access and implementation process for 3rd parties.

2. Understanding the FHIR API structure

The FHIR API is created from a standard framework of resources.

Types of FHIR API resources
Resource Definition
CodeSystem A collection of codes also known as ontology or terminology, in this case SNOMED CT
ValueSet A specific subset of codes from the CodeSystem, for example all codes relating to influenza
ConceptMap Defines the relationship between concepts, including from one CodeSystem to another

You can find more information about resources and how to use them on the HL7 website, including examples and descriptions.

3. Making API requests

You make API requests using operations like $lookup, $expand and $translate, which are specific to each resource. View the full list of available operations.

Expanding a ValueSet

The most common request users have is to search and retrieve specific SNOMED CT codes. You can do this by using the $expand a ValueSet operation.

The rules of this operation can be broad, like retrieving all codes related to influenza. They can also be very specific, like retrieving data created on a specific date.

See some example $expand requests in action in a Postman environment on the Ontoserver website.

4. Getting support

If you need integration support, help is available.

For specific technical questions, you may want to join the FHIR community. This FHIR knowledge-sharing website allows you to search through content posted by other users, and to post your own issues and questions.

For more general support with the terminology service, email terminology.service@wales.nhs.uk.
