API catalogue

  • User authentication and authorisation

    Live SOAP

    Central Audit Service (CAS)

    Write pre-defined audit events for AC3 users that are compliant with audit requirements and make audit logs available to other systems.

  • Patient address search and match

    Live SOAP

    Welsh Address Matching Service (WAMS)

    Search for addresses in the nationally approved address list, standardising manually entered addresses to be consistent with other systems.

  • Birth registration

    Live SOAP

    Welsh Birth Notification Service (WBNS)

    Register a birth and receive a newly generated NHS number.

  • Clinical document exchange

    Live SOAP

    Welsh Clinical Communications Gateway (WCCG)

    Send and receive clinical documents, such as referral letters. Also known as clinical documents exchange.

  • Patient document storage

    Live SOAP

    Welsh Care Record Service (WCRS)

    Store documents such as referrals in WCRS in formats including XML, PDF and Word with metadata.

  • Patient document advanced search

    Live SOLR

    Welsh Care Record Service (WCRS)

    Perform an advanced search of WCRS to find patient documents using Apache Solr. Group and filter results by category to find documents more quickly.

  • Reference data retrieval (FHIR)

    Experimental Sandbox FHIR

    Welsh Reference Data and Terminology Service (WRTS)

    Retrieve organisational reference data including lists of practitioners, GP practices and hospitals using FHIR resources.

  • Terminology search (FHIR)

    Experimental Sandbox FHIR

    Welsh Reference Data and Terminology Service (WRTS)

    Search and read clinical terminologies like SNOMED CT using FHIR resources.

  • Patient demographic information search and match

    Live SOAP

    Master Patient Index (MPI)

    Search for patient identifying information matched across multiple systems, such as NHS number, sex, date of birth and address.

  • Patient demographic information search

    Live SOAP

    Welsh Demographics Service (WDS)

    Search for patient identifying information, such as NHS Number, name and date of birth. A standard exists for synchronising this data with a system's locally stored patient record.

  • PROMS event questionnaire request

    Live SOAP

    Patient Reported Outcome Measures (PROMS)

    Request an assessment questionnaire for a patient after a clinical event at relevant stages along a treatment pathway.

  • Patient document storage (FHIR)

    Experimental Sandbox FHIR

    Welsh Care Records Service (WCRS)

    Store documents in WCRS using the FHIR DocumentReference resource.

  • Patient test results search (FHIR)

    Experimental Sandbox FHIR

    Welsh Results Reporting Service (WRRS)

    Search and retrieve a patient's test results using the FHIR DiagnosticReport and Observation resource.

  • Patient identifer cross-reference (PIX)

    Live SOAP

    Master Patient Index (MPI)

    Look up alternative local identifiers for a patient from multiple systems using the Master Patient Index.

  • Reference data and schema lookup

    Live REST

    Welsh Reference Data Service (WRDS)

    Retrieve table schemas and data listing all fields in the database.

  • Reference data standard lookup

    Live SOAP

    Welsh Reference Data Service (WRDS)

    Retrieve specific fields from reference sets, such as location codes for hospitals, lists of available drugs and pathology tests.

  • RPatient GP record retrieval

    Live SOAP

    Welsh Reference Data Service (WRDS)

    Access patient data held on GP systems, such as medications, diagnoses and test results.

  • Patient test results request and report

    Live SOAP

    Welsh Results Reporting Service (WRRS)

    Request radiology scans and pathology tests on specimens like blood samples.
