Accessibility

We use government standards for accessibility alongside the Web Content Accessibility Guidelines (WCAG). WCAG is widely accepted as the international standard for accessibility on the web.

Although we aim to make this website accessible to all users and achieve a WCAG conformance level of AA, we continually work with stakeholders to ensure that conformance level A is met as a minimum.

Contact us by email if you experience accessibility issues on this website.

Use text-to-speech (TTS)

Adobe Read Out Loud

Use the built-in Read Out Loud function in Adobe Reader (version 7 and above) to hear the content in a PDF.

To turn it on:

  1. Open a PDF document and click on View
  2. Select Read Out Loud from the menu
  3. Select Activate Read Out Loud

Read Out Loud keyboard shortcuts
Keyboard shortcut Action
Shift + Ctrl + Y Activate Read Out Loud
Shift + Ctrl + V Read current page
Shift + Ctrl + B Read to the end of the document
Shift + Ctrl + C Pause or resume reading
Shift + Ctrl + E Stop reading the document

Change text size

You can adjust the text in your browser to make content clearer and easier to read.

Change text size on desktop

In Google Chrome, Microsoft Edge and Firefox:

  • select the settings menu, which is the three dots in the top right corner of your browser
  • change text size by selecting Zoom - or +

Or use keyboard shortcuts (any browser):

  • press Ctrl + '+' to increase text size
  • press Ctrl + '-' to decrease text size

Change text size on mobile

On your iPhone:

  • open your Settings app
  • select Accessibility from the menu
  • select Display & Text size from the menu
  • use the slider to choose your text size

On your Android device:

  • open your Settings app
  • select Accessibility from the menu
  • select Font size from the menu
  • use the slider to choose your text size

Download readers and viewers

You may need to download some free readers and viewers to access different document formats:

  • download Adobe Acrobat Reader (opens in a new window) to view PDFs
