Get started

1. Explore

You can find information about NHS Wales APIs in the catalogue to discover what suits your needs

1. Archwilio

Gallwch ddod o hyd i wybodaeth am APIs GIG Cymru yn y catalog i ddarganfod beth sy'n addas i'ch anghenion

2. Sign up

Sign-up to create a user profile. You'll receive an email to verify your email address.

2. Cofrestrwch

Cofrestrwch i greu proffil defnyddiwr. Byddwch yn derbyn e-bost i wirio'ch cyfeiriad e-bost.

3. Subscribe

Before making calls to APIs you'll need to subscribe to our Sandbox product, which currently contains some experimental FHIR APIs. We hope to soon make sandbox implementations of other APIs from the NHS Wales catalogue.

3. Tanysgrifio

Cyn gwneud galwadau i APIs bydd angen i chi danysgrifio i'n cynnyrch Sandbox, sydd ar hyn o bryd yn cynnwys rhai APIs FHIR arbrofol. Gobeithiwn yn fuan i roi APIs eraill ar waith mewn blwch tywod o gatalog GIG Cymru.

4. Try it out

Once you've subscribed, navigate to your user profile to see a list of subscription keys. You'll need to include the subscription key in the HTTP header when making API requests. On the API pages you can click the "Try it" button to test it out in the browser.

4. Rhowch gynnig arni

Ar ôl i chi danysgrifio, llywiwch i'ch proffil defnyddiwr i weld rhestr o allweddi tanysgrifio. Bydd angen i chi gynnwys yr allwedd tanysgrifio ym mhennyn HTTP wrth wneud ceisiadau API. Ar y tudalennau API gallwch glicio ar y botwm "Rhowch gynnig arni" i'w brofi yn y porwr.
