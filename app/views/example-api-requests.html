Example API requests

Information:

You must subscribe to the sandbox (opens in new tab) before you try making an API request.

GET ValueSet/$expand (most common request)

Search and retrieve specific SNOMED CT codes by expanding a ValueSet.

Request

GET https://sandbox.api.nhs.wales/wrds/fhir-terminology/ValueSet/$expand[?url][&filter][&count]



Request parameters
Name In Required Type Description
Name url In query Required false Type string Description
Name filter In query Required false Type string Description
Name count In query Required false Type integer Description


Response: 200 OK

Success

GET CodeSystem

Information about the request

View request and parameters

Request

GET https://sandbox.api.nhs.wales/wrds/fhir-terminology/CodeSystem[?_id][&_tag][&description][&identifier][&name][&system][&url][&version][&_format]



Request parameters
Name In Required Type Description
Name _id In query Required false Type string Description The ID of the resource
Name _tag In query Required false Type string Description
Name description In query Required false Type string Description The description of the code system
Name identifier In query Required false Type string Description External identifier for the code system
Name name In query Required false Type string Description Computer-friendly name of the code system
Name system In query Required false Type string Description The system for any codes defined by this code system (same as 'url')
Name url In query Required false Type string Description The uri that identified the code system
Name version In query Required false Type string Description The business version of the code system
Name _format In query Required false Type string Description


Response: 200 OK

Success
